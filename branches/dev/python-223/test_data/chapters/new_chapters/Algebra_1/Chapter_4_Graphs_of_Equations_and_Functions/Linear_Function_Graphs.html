Introduction – Functions

So far we have used the term function to describe many of the equations we have been graphing, but the concept of a function is extremely important in mathematics. Not all equations are functions. In order to be a function, the relationship between two variables, and , must map each value to exactly one value.

Visually this means the graph of versus must pass the vertical line test meaning that a vertical line drawn through the graph of the function must never intersect the graph in more than one place.
