Introduction

Recall that we used different ways to approximate the value of integrals. These included Riemann Sums using left and right endpoints, as well as midpoints for finding the length of each rectangular tile. In this lesson we will learn two other methods for approximating integrals. The first of these, the Trapezoidal Rule, uses areas of trapezoidal tiles to approximate the integral. The second method, Simpson’s Rule, uses parabolas to make the approximation.

It is reasonable to think that other methods of approximating curves might be more applicable for some functions. Simpson’s Rule is a method that uses parabolas to approximate the curve.
