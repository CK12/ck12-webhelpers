Identify Functions Using Differences or Ratios.

One method for identifying functions is to look at the difference or the ratio of different values of the dependent variable.

We use differences to identify linear functions.

If the difference between values of the dependent variable is the same each time we change the independent variable by the same amount, then the function is linear.
