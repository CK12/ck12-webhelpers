Introduction

So far you know how to solve quadratic equations by factoring. However, this method works only if a quadratic polynomial can be factored. Unfortunately, in practice, most quadratic polynomials are not factorable. In this section you will continue learning new methods that can be used in solving quadratic equations. In particular, we will examine equations in which we can take the square root of both sides of the equation in order to arrive at the result.
