{{ _("About the eFlex") }}

{{ _("What is eFlex?") }}

{{ _("eFlex is an email interface to CK-12's FlexBook® textbook authoring system by which one can send request and get dynamically collated book out of the sent request in various digital formats such as PDF, Mobi, EPUB.") }}

{{ _("How to Use eFlex?") }}

{{ _("You can use this interface to:") }}

  1. {{ _("Create a new FlexBook® textbook") }}
  2. {{ _("Contribute a Lesson") }}

{{ _("Creating a New FlexBook® textbook") }}

{{ _("To create a new FlexBook® textbook, please send an email to") }} {{ _("eflex@ck12.org") }}{{ _(", with:") }}

  1. {{ _('A subject line in the format of "Create: Book Title"') }}
  2. {{ _('And mail body containing one or more lines in the format of "Subject: topic keywords", for the book content.') }}
  3. {{ _("(Optional in the mail content): @Forward") }}<{{ _("digital format") }} >: <{{ _("list of Email IDs") }}>
    {{ _("Digital format can be pdf, epub or mobi.") }}

{{ _("Example:") }}

{{ _('The following is an email for requesting a book titled "My Small Physics Book" containing 2 concepts; the first covering gravity, and the second covering optics. Also, the EPUB version of the resulting book will be forwarded to the specified Email IDs.') }}

{{ _("Subject line:") }} {{ _("Create: My Small Physics Book") }}
{{ _("Mail body:") }} {{ _("Physics: gravity") }}
{{ _("Physics: optics") }}
{{ _("@Forwardepub: friend1@maildomain.com, friend2@maildomain.com") }}
