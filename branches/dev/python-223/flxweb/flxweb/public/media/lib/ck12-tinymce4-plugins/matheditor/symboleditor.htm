How to use the Math Editor?
Alignat    
HWP math uses alignat setting which allows aligning of multiple equations better while keeping the inline style.
Inline    
Inline math allows math to be inserted within the text without breaking the text of the flow.
Blockmath  
Block math is used for multiline equations. They are usually centered on the page width. No text or other entities can appear on the same lines as block math.
